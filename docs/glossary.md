# 用語集

| 用語 | 意味 |
|---|---|
| 翌日準備 | 翌日の席割、担当割、帳票作成、印刷などの準備業務 |
| 販売在庫チェック | PMS・手間いらず・JTB間の販売在庫を照合し、差異を確認する業務 |
| 正式ソース | `Development\repos` 配下にある、開発・ビルド・Git操作の起点となるリポジトリ |
| 共有版 | 共有フォルダに配布され、現場で使用されるアプリケーション一式 |
| デモ機 | 実運用前の確認や説明に使用する端末。実運用機との設定差に注意する |
| 休館日 | 在庫照合や警告判定から除外すべき、施設が営業しない日 |
| 担当割 | スタッフシフト等を基に、業務担当を割り当てること |
| 配布物 | `DinnerSystem.exe`、`_internal` など、ビルドして共有版へ反映するファイル |
| 業務データ | 顧客情報、取込CSV、保存データ、出力、現場設定など、運用中に生成・利用されるGit管理外データ |
| `_internal` | PyInstallerで生成されるPythonランタイムと依存ライブラリの領域。共有版更新時は完全同期する |
| 安定版 | 必要な確認を終え、実運用に使用できるとしてタグ付けした版 |
